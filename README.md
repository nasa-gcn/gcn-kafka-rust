# GCN Kafka Client for Rust

[![crates.io](https://img.shields.io/crates/v/gcn-kafka.svg)](https://crates.io/crates/gcn-kafka)

This is the official Python client for the [General Coordinates Network (GCN)](https://gcn.nasa.gov). It is a very lightweight wrapper around [rdkafka](https://crates.io/crates/rdkafka).
